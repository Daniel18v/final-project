Vamos a realizar una página web de videojuegos con una pequeña red social interior para poder compartir juegos creados por y para la comunidad y ponerles un precio (Market de aplicaciones).

La página va a tener sus propios videojuegos y un apartado en el que se "scrapearán" videojuegos de otras plataformas con información  relacionada, y enlaces a los mismos.

Además de eso, haremos un chat para la comunidad para que se puedan compartir y realizar preguntas (Véase intercambiosvirtuales.org).

En la parte de administración se tendrá acceso a la base de datos para administrar artículos y gestión de usuarios, así como cualquier tema de administración (visitas, estadísticas, etc...).

Frameworks utilizados y librerías: Phaser.io, jQuery, jQuery UI, Node.js, Express, Socket.io.
