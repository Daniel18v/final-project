The helper application is created using

TMS Software TAdvSmoothLabel component

You can get a trial version here: https://www.tmssoftware.com/site/advsmoothlabel.asp


==========================


If you do not have this component, you can easily remove it by editing uMain.pas like so:

removing "AdvSmoothLabel" from uses list

Replacing all instances of TAdvSmoothLabel with TLabel or similar component.

These components are: lblCountDown, lblHeader, lblFooter.

Update lines in FormCreate procedure where text is assigned to these components.
